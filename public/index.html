Discography Timelines is a personal project aiming to present every album released by an artist on a timeline (based on the release date of the album).

The app allows users to add the discography of different artists of their choice by consuming the Spotify Web API.

From a technical point of view, the goal was to build a front-end only website based on a frameworkless philosophy. This project has therefore been built with HTML, CSS and Vanilla Javascript only.
